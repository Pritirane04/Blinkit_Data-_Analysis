Blinkit SQL Data Analysis Project

This project focuses on analyzing the dataset of Blinkit Blinkit using SQL. The aim is to extract meaningful business insights from raw data such as sales, product categories, customer orders, and delivery performance.

To analyze grocery order and sales data
To identify top-performing products and categories
To understand customer purchasing behavior
To improve decision-making using SQL queries
